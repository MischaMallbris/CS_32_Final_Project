# CS_32_Final_Project
I am creating a choose your own adventure game. An evil wizard has trapped you inside a computer and you have to escape his
digital maze by making choices on where to go and solving some riddles along the way.

To run the code please intialize the code by clicking the play button, and then afterwards type "streamlit run game32.py"
in the terminal. Because the game is in python streamlit, this will open a new tab within your browser where you will be
able to play the game.

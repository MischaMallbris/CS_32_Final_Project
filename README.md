# CS_32_Final_Project

**Summary Story:**
A choose your own adventure game! Are you ready? An evil wizard has trapped you inside a computer and you have to escape his digital maze by making choices on where to go and solving some riddles along the way. Can you find your way out without being deleted?

Consists of 10 doors and 4 possible endings.

**How to Run:**
Make sure you have Python 3.8+ installed. Then install Streamlit: bash: pip install streamlit
No other packages are required. All four project files must be in the same folder

To run the code please intialize the code by clicking the play button, and then afterwards type "streamlit run game32.py"
in the terminal. Because the game is in python streamlit, this will open a new tab within your browser where you will be
able to play the game. If it doesn't, navigate to the URL shown in the terminal (usually http://localhost:8501)

**Additional Notes:**
Sound effects require a browser that supports the Web Audio API (all modern browsers do). Click any button to unlock audio on first load, this is a browser security requirement.
No API keys, file uploads, or external services are needed.
An internet connection is required to load Google Fonts and the Unsplash scene images.
